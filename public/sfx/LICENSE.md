# Sound Effects License

Sound effects used in this project should be placed in this directory.

## Recommended sources

### taira-komori.net (家長指定)

- URL: https://taira-komori.net/freesoundtw.html
- License: フリー素材（Free-to-use, including commercial use）
- Attribution: 効果音ラボ / Taira Komori
- Required files:
  - `tap.mp3` — button click
  - `pop.mp3` — bubble pop
  - `correct.mp3` — correct answer chime
  - `wrong.mp3` — wrong answer buzz
  - `levelup.mp3` — level complete fanfare
  - `star.mp3` — star earned
  - `combo.mp3` — combo streak
  - `page_turn.mp3` — page / scene transition

## Notes

- All audio files are loaded lazily; missing files are silently ignored at runtime.
- Place `.mp3` files directly in `public/sfx/` (no subdirectory needed).
- Verify the individual license of each downloaded file before distribution.
